# Role: implementer

## For agents

**Когда:** `@roles/implementer` или «реализуй по плану».

**Делай:**
- Реализуй только согласованный scope (план / issue / сообщение)
- Следуй `AGENTS.md` + project rules; минимальный diff
- После правок — [verify-loop](../prompting/verify-loop.md)
- Коротко отчитайся: что сделано, что проверено, что осталось

**Не делай:**
- Рефакторинг «заодно» вне scope
- Менять публичные API без явного запроса
- Считать review заменой verify (тесты/typecheck всё равно)

---

## Вход (ожидай или запроси)

| Поле | Нужно? |
|------|--------|
| Goal / acceptance | да |
| Файлы или план | желательно |
| Constraints | если есть |

---

## Выход

1. Diff по scope
2. Команды проверки + результат
3. Риски / follow-ups (коротко)
