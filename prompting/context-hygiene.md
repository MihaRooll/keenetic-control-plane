# context-hygiene

> Паттерн: минимальный релевантный контекст. Источник идей: [`docs/cursor-agent-best-practices.md`](../docs/cursor-agent-best-practices.md).

## For agents

**Когда:** агент «плывёт»; смена фичи; длинный тред; пользователь кидает много `@` без нужды.

**Применяй:**
- Не знаешь файл → опиши задачу, ищи сам; не проси весь репо в контекст
- Знаешь точный файл → `@` его
- Смена задачи / путаница → **new chat**
- Нужен прошлый контекст → `@Past Chats` / `@Branch`, не копипаста треда
- Rules/skills: коротко; детали в файлах по ссылке

**Не делай:** пихать irrelevant файлы «на всякий»; продолжать мёртвый длинный чат; дублировать style guide в каждое сообщение.

## Таблица действий

| Ситуация | Действие |
|----------|----------|
| Знаешь путь | `@file` |
| Не знаешь | Описание + search |
| Смена фичи | New chat |
| Нужна история | `@Past Chats` / `@Branch` |
| Шум после summarize | New chat + короткий brief |

## Brief для нового чата (шаблон)

```
Задача: …
Контекст: @key-file (и только нужное)
Ограничения: …
Done when: …
```
